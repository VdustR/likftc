# likftc Documentation

## Development

```bash
# Install dependencies
pnpm i

# Start dev server:
pnpm dev

# Build Production
pnpm build

# Build Production With BASE_URL
BASE_URL=/likftc pnpm build
```

## Update Demo Code

```bash
pnpm gen-code
```

## Build Sitemap

You have to build sitemap after build production:

```bash
pnpm sitemap
```
